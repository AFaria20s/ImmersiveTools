## About the Mod

**Vanilla Enhancer: Ores** ‚Äî also known as **Vancer Ores** ‚Äî is an expansion to the Minecraft vanilla experience that adds new ores, tools, and armor sets, all while keeping the original game‚Äôs balance and aesthetics.

### Features

- **Amethyst Ore**
  - Naturally generates in the Overworld.  
  - Includes a complete **Amethyst Toolset** and **Armor Set**.

- **Enderium Ore** 
  - Spawns exclusively in the End dimension.  
  - Includes **Enderium Tools** and **Armor**, with enhanced durability and power.

## Download

### For Players

1. Download and install **[Minecraft Forge 1.16.5](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.16.5.html)**.  
2. Download the latest version of **Vancer Ores** from the [Releases](https://github.com/AfonsoFaria20s/Vanilla-Enhancer-Ores/releases) page.  
3. Place the `.jar` file inside your Minecraft `mods` folder:  
   ```
   .minecraft/mods
   ```
5. Launch the game using the **Forge** profile.

## Developer Setup (Manual Installation)

If you want to contribute or modify the mod locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AfonsoFaria20s/Vanilla-Enhancer-Ores.git
   cd Vanilla-Enhancer-Ores
   ```

2. **Import the project** into your IDE (e.g. IntelliJ IDEA or Eclipse) as a **Gradle project**.

3. **Run Gradle setup tasks:**
   ```bash
   ./gradlew genEclipseRuns      # for Eclipse
   ./gradlew genIntelliJRuns     # for IntelliJ IDEA
   ./gradlew build
   ```

4. Once the environment is set up, you can start the client directly from your IDE to test the mod.
